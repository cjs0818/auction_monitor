# 온비드 시·도 전체검색 재수정

## 문제
차세대 온비드 API에서 `경기도`처럼 시·도만 지정한 요청이 0건을 반환할 수 있다.
이전 보완 로직은 전국 대체조회도 0건이면 이를 완전한 조회로 판단하여 시·군·구 분할조회를 생략할 수 있었다.
그 결과 `경기도 가평군` 개별검색에서는 물건이 나오지만 `경기도` 전체검색에서는 0건이 되는 문제가 남아 있었다.

## 수정
- 시·도 단독조회 결과가 0건이면 전국조회 결과에 의존하지 않는다.
- 선택한 시·도의 공식 시·군·구를 모두 직접 조회한다.
- 각 시·군·구 응답은 주소를 다시 검증해 타 지역 혼입을 제거한다.
- 동일 물건의 공매 회차는 기존 방식대로 병합한다.
- 경기도는 31개 시·군을 순회하므로 가평군 개별검색에서 확인되는 물건도 경기도 전체검색에 포함된다.

## 진단
검색 진단에 `차세대 온비드 시·군·구 전수 분할검색`으로 표시된다.
