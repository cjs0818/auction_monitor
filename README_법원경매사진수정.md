# 법원 경매 대표사진 수집 수정

## 원인

법원경매 물건목록·사건상세 JSON에는 대표사진 URL이 포함되지 않습니다. 기존 카드 화면은 원시 JSON 안의 이미지 주소만 찾았기 때문에 온비드 공매 사진만 표시되고 법원 경매는 자리표시자로 나타났습니다.

## 수정 방식

- 검색조건을 통과한 후보를 먼저 평가하여 화면에 표시할 상위 물건을 선정합니다.
- 상위 법원 경매물건만 공식 법원 사건검색 상세화면을 엽니다.
- `물건사진`, `현황사진`, `사진보기` 등의 화면 요소와 팝업을 찾아 가장 큰 물건사진을 PNG로 캡처합니다.
- 캡처한 사진은 `data/court_photo_cache/`에 보관하고 다음 검색부터 재사용합니다.
- 공매는 기존처럼 온비드 API의 썸네일을 사용합니다.
- 사진을 찾지 못해도 검색과 보고서 작성은 계속 진행합니다.

## 설정

대시보드의 `수집·운영 설정 → 법원 경매 대표사진`에서 다음을 조정할 수 있습니다.

- 법원 상세화면에서 대표사진 자동수집
- 1회 실행 사진 신규수집 상한
- 사진 캐시 유지일수
- 사진 캐시 비우기

첫 검색은 사건별 상세화면을 추가로 열기 때문에 시간이 늘어날 수 있습니다. 이후에는 캐시를 사용하므로 사진 표시가 빠릅니다.
