# 제주특별자치도 시·도 전체검색 HTTP 0 수정

## 문제

제주특별자치도 `시도전체` 검색을 제주시(`50/110`)와 서귀포시(`50/130`)로 연속 분할 호출할 때 법원 사이트의 브라우저 내부 `fetch`가 간헐적으로 `status 0 / TypeError: Failed to fetch`를 반환하며 전체 검색이 중단될 수 있었습니다.

## 수정

- 제주도 전체검색은 제주지방법원 코드 `B000530`으로 한 번만 조회합니다.
- 지역코드는 비우고 법원 전체 결과에서 제주특별자치도 주소만 유지합니다.
- `status 0` 발생 시 공식 검색화면 세션을 다시 준비하고 동일 요청을 한 번 자동 재시도합니다.
- 검색 진단에는 `제주지방법원 전체검색 후 주소검증`으로 표시됩니다.

## 검증

- 제주도 전체 요청 본문: `cortOfcCd=B000530`, 지역코드 공란, `cortStDvs=1`
- 제주시·서귀포시 결과 병합 테스트
- `Failed to fetch` 1회 발생 후 세션 복구·재시도 테스트
- 전체 자동 테스트 116개 통과
