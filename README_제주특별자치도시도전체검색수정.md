# 제주특별자치도 시·도 전체 경매검색 수정

## 문제

`제주특별자치도`를 시·도 전체로 선택하면 법원경매 결과와 제외물건이 모두 0건으로 표시될 수 있었습니다.

기존 시·도 전체검색은 법원 사이트의 시·도 단독 요청 오류를 피하기 위해 전국 목록을 조회한 뒤 주소로 대상 시·도를 걸러냈습니다. 그러나 전국 결과가 많은 기간에는 제한된 앞쪽 페이지만 수집하므로 제주 물건이 뒤 페이지에 있으면 한 건도 확보하지 못할 수 있었습니다.

## 수정

제주특별자치도 전체검색을 다음 두 개의 정확한 시 단위 요청으로 분할합니다.

- 제주시: `rprsAdongSdCd=50`, `rprsAdongSggCd=110`
- 서귀포시: `rprsAdongSdCd=50`, `rprsAdongSggCd=130`

두 요청 결과를 합친 뒤 동일 사건·물건을 병합하고, 최종적으로 제주특별자치도 주소를 다시 검증합니다.

시·군·구 수가 8개 이하인 시·도는 같은 방식으로 정확검색하며, 시·군·구가 많은 시·도는 호출량을 고려해 기존 대체검색을 유지합니다. 기준값은 `province_fanout_max_municipalities`에서 조정할 수 있습니다.

## 검증

- 제주도 전체 선택 시 전국 요청을 사용하지 않는지 확인
- 제주시와 서귀포시 요청이 모두 실행되는지 확인
- 제주시에서 결과가 나와도 서귀포시 조회가 생략되지 않는지 확인
- 전체 자동 테스트 115개 통과
