# 지역 우선검색 수정

## 핵심 수정

- 시·군·구 선택 시 법원 요청의 `rprsAdongSggCd`에 5자리 행정표준코드가 아니라 법원 화면이 사용하는 3자리 시군구코드를 전송합니다.
  - 전북특별자치도 부안군: `52 / 800`
  - 과거 전라북도 코드 병행: `45 / 800`
- 시·군·구 직접검색에서 대상지역 물건이 확인되면 시·도 전체 대체검색을 생략합니다.
- 직접검색이 0건일 때만 시·도 대체검색을 수행합니다.
- 시·도 대체검색 응답은 즉시 주소를 검증하여 선택한 시·군·구 물건만 수집합니다.
- 다른 시·군·구 물건은 runner에 전달하지 않으므로 `수집됐지만 지역 불일치` 목록에 나타나지 않습니다.
- 검색 진단에 `지역일치건수`와 `지역불일치제외`를 표시합니다.

## 적용 후 예상 요청

전북특별자치도 부안군 선택 시 첫 요청은 다음과 같습니다.

```text
rprsAdongSdCd  = 52
rprsAdongSggCd = 800
cortStDvs      = 2
```

현재 코드로 대상 물건을 찾지 못하면 과거 도 코드 `45 / 800`을 조회합니다.
두 정확검색 모두 대상지역 0건일 때만 전북특별자치도/전라북도 전체 대체검색을 수행하고, 응답 중 부안군 주소만 보존합니다.
