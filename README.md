# LandWatch — 토지 경매·공매 통합 모니터

대한민국 법원경매정보의 **법원경매**와 한국자산관리공사 온비드의 **공매**를 한 화면에서 검색하고, 같은 지역·가격·면적·유찰·용도 조건으로 투자후보를 평가하는 macOS용 로컬 프로그램입니다.

대시보드 첫 화면에서 다음 중 하나를 선택합니다.

- `경매`: 대한민국 법원경매정보만 조회
- `공매`: 한국자산관리공사 온비드 공매만 조회
- `경매 및 공매`: 두 출처를 순서대로 조회하여 하나의 결과표·보고서로 통합

## 주요 기능

- 전국 시·도 및 시·군·구 다중선택
- 신건·진행·유찰·재매각·수의계약·입찰마감 상태 선택
- 전·답·과수원·임야 등 토지용도 선택
- 최저가격, 감정평가액, 할인율, 유찰횟수, 면적, 일정 범위 지정
- 투자검토점수, 위험조건 제외, 신규·변경 감지
- 사건번호 또는 공고번호 링크를 새 탭에서 열어 상세정보·네이버 지도 표시
- 경매·공매 통합 CSV/HTML 보고서
- Telegram·이메일 알림
- 동일조건 캐시와 매일 자동 실행

## 데이터 수집 방식

### 법원경매

Selenium으로 대한민국 법원경매정보 브라우저 세션을 생성하고, 브라우저 내부에서 사이트의 물건검색 요청을 읽기 전용으로 실행합니다. CAPTCHA·보안문자·접근차단은 우회하지 않습니다.

### 온비드 공매

공공데이터포털의 `캠코공매물건조회서비스` 인증키를 사용합니다. 온비드 OpenAPI에서 다음 조건을 서버에 직접 전달합니다.

- 매각 방식
- 부동산·토지 카테고리
- 시·도 및 시·군·구
- 감정평가액과 최저입찰가
- 입찰 시작·마감 기간

공매 결과는 공고번호, 공매번호, 물건번호, 물건관리번호, 소재지, 최저입찰가, 감정평가액, 유찰횟수, 입찰마감일을 경매 결과와 같은 표준 항목으로 변환합니다.

## 1. macOS 설치

```bash
cd land_auction_monitor
./scripts/setup_mac.sh
```

Conda·ROS2 환경과 충돌했던 Mac에서는 다음을 사용합니다.

```bash
./scripts/repair_mac_environment.sh
```

대시보드 실행:

```bash
.venv/bin/python -I -m streamlit run app.py
```

또는 Finder에서 `대시보드실행.command`를 더블클릭합니다.

## 2. GUI에서 공매 인증키 등록

1. 공공데이터포털에 회원가입합니다.
2. `캠코공매물건조회서비스`를 활용 신청하고 일반 인증키를 발급받습니다.
3. LandWatch 대시보드에서 `수집·알림 설정`을 엽니다.
4. `온비드 공매 설정 → 공공데이터포털 서비스 인증키`에 입력합니다.
5. `수집 설정 저장`을 누릅니다.

설정파일을 직접 편집할 필요가 없습니다. 환경변수를 선호하면 다음도 지원합니다.

```bash
export KAMCO_API_KEY='발급받은 일반 인증키'
```

## 3. 검색 실행

1. `검색조건 관리`에서 지역·상태·용도·가격·유찰횟수·면적을 저장합니다.
2. `검색·대시보드`에서 검색대상을 선택합니다.
3. `연결 점검`을 누릅니다.
4. `지금 검색 실행`을 누릅니다.

공매를 선택했는데 인증키가 없으면 대시보드가 등록 위치를 안내합니다. 경매 및 공매를 선택하면 두 출처를 모두 조회한 뒤 결과의 `매각구분` 열에 `경매` 또는 `공매`가 표시됩니다.

## 4. CLI 실행

```bash
# 경매만
.venv/bin/python -I -m landwatch.cli run --target 경매 --no-notify

# 공매만
.venv/bin/python -I -m landwatch.cli run --target 공매 --no-notify

# 경매와 공매 통합
.venv/bin/python -I -m landwatch.cli run --target "경매 및 공매" --no-notify

# 공매 연결 점검
.venv/bin/python -I -m landwatch.cli connection-check --target 공매
```

## 5. 결과표 표준항목

| 공통 항목 | 법원경매 | 온비드 공매 |
|---|---|---|
| 매각구분 | 경매 | 공매 |
| 번호 | 사건번호 | 공고번호 |
| 물건식별번호 | 물건번호 | 물건관리번호 |
| 진행기관 | 관할법원 | 입찰주최기관 |
| 최저가격 | 최저매각가격 | 최저입찰가 |
| 일정 | 매각기일 | 입찰마감일 |
| 상세 원문 | 법원경매정보 | 온비드 |

## 6. 상세페이지

검색 결과의 사건번호 또는 공고번호를 클릭하면 새 탭이 열립니다.

- 기본정보와 투자검토점수
- 가격·할인율·면적·유찰횟수
- 네이버 지도
- 법원 사건정보 또는 온비드 공매정보
- 입찰일정과 원시 응답
- 원문 열기

## 7. 자동 실행

기본 매일 오전 7시 30분:

```bash
./scripts/install_mac_launchd.sh
```

다른 시간 예시:

```bash
./scripts/install_mac_launchd.sh 8 10
```

자동 실행은 대시보드에서 마지막으로 저장한 검색대상을 사용합니다.

## 8. 캐시와 오류자료

- 법원 검색 캐시: `data/selenium_cache/`
- 법원 오류자료: `data/selenium_debug/`
- 온비드 검색 캐시: `data/onbid_cache/`
- SQLite 이력: `data/landwatch.db`
- 보고서: `reports/`

대시보드의 `수집·알림 설정`에서 경매·공매 캐시를 각각 비울 수 있습니다.

## 9. 주의사항

자동 점수는 후보 압축용입니다. 참여 전 반드시 원문과 첨부서류를 다시 확인하십시오.

- 권리관계, 점유 및 인수조건
- 공유지분, 법정지상권, 분묘기지권, 유치권
- 도로 접면, 경계, 맹지와 실제 진입로
- 토지이용계획 및 개발행위 제한
- 농지취득자격증명 발급 가능성
- 온비드 재산구분·처분조건·입찰보증금·매각결정 조건
- 현장과 인근 실거래 시세
