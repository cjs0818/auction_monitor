# 법원 일괄매각 목적물 병합 수정

## 문제
법원 조건검색 응답의 `mokmulSer`(목적물번호)를 실제 경매 물건번호로 사용하여,
일괄매각 사건의 토지·건물 목록이 각각 독립된 경매물건처럼 표시되었습니다.
그 결과 한 물건 전체의 감정가와 최저매각가격이 모든 목적물 행에 반복되었습니다.

## 수정
- 실제 물건순번인 `maemulSer`를 우선 사용합니다.
- `mokmulSer`는 목적물번호로 별도 보존합니다.
- 같은 법원·사건번호·물건순번의 목적물은 한 경매물건으로 병합합니다.
- 주소는 `대표 소재지 외 N건`, 토지면적은 목적물별 면적 합계로 표시합니다.
- 감정가와 최저매각가격은 물건 전체 가격이므로 합산하지 않고 한 번만 유지합니다.
- 현행/과거 지역코드 및 날짜구간 중복 조회에서도 목적물 면적이 중복 합산되지 않습니다.

## 검증 사례
`순천지원 2025타경50039`의 155개 목적물은 물건번호 1의 한 일괄매각 물건으로 병합되며,
토지면적 합계는 271,219㎡, 감정가 18,598,941,000원, 최저매각가격 8,326,233,000원입니다.
