# 경매·공매 통합검색 숫자 변환 오류 수정

## 증상

`경매 및 공매` 선택 시 다음 오류가 발생했습니다.

```text
invalid literal for int() with base 10: '-'
```

## 원인

개별 공급자의 진단값에서 해당되지 않는 요청 건수를 `-`로 표시하고 있었고,
통합 공급자가 경매·공매 진단을 합산하면서 이를 정수로 직접 변환했습니다.

## 수정

- 통합 진단 합산 시 `-`, 빈 문자열, `None`을 0으로 안전하게 정규화
- 경매 공급자의 공매 요청 수와 공매 공급자의 법원 요청 수를 숫자 0으로 기록
- 혼합형 진단값 회귀 테스트 추가
