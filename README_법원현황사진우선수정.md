# 법원경매 현황사진 우선 표시 수정

## 변경 목적

법원경매 카드에 현황사진 대신 감정평가서·현황조사서 등의 법원문서 또는 사건화면의 위치지도가 대표 이미지로 선택되는 문제를 수정했습니다.

## 변경 내용

1. 현황사진/물건사진 탭을 먼저 열고 해당 사진만 대표 이미지로 저장합니다.
2. 사진 탐색 순서는 `현황사진 → 물건사진 → 사진보기 → 사진정보 → 물건이미지 → 사진`입니다.
3. 사건검색 화면이나 물건내역 화면의 가장 큰 이미지·canvas를 무조건 저장하던 처리를 제거했습니다.
4. 감정평가서, 현황조사서, 매각물건명세서 등 법원문서를 대표 이미지로 사용하는 처리를 완전히 제거했습니다.
5. 사진 후보에서 PDF·문서뷰어·지도 관련 DOM 요소를 제외합니다.
6. 현황사진이 없으면 네이버 위치지도를 대표 이미지로 저장합니다.
7. 과거 버전에서 저장된 `court-document-*`, `court-case-page`, `court-item-row`, `court-item-detail`, `address-fallback` 캐시는 자동 폐기하고 다시 수집합니다.
8. 카드 우측 상단 표시는 `현황사진` 또는 `위치지도`로만 구분합니다.

## 적용 후 참고

기존 검색결과가 메모리에 남아 있으면 새로 검색해야 교체된 이미지가 표시됩니다. 캐시 출처가 구버전 형식이면 다음 상세 보강 시 자동으로 삭제 후 재수집됩니다.


## 추가 수정
- 현황사진이 없을 때는 위치지도를 더 이상 표시하지 않고 카드 이미지를 비워 둡니다.
- 기존 `naver-map-fallback` 캐시는 자동 무효화됩니다.
