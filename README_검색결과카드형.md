# 검색결과 사진 카드형 화면

첫 화면의 검색결과를 단순 표 대신 사진과 핵심정보 중심의 카드 목록으로 표시합니다.

## 카드 표시 항목

- 대표 물건사진
- 매각기일 또는 입찰마감일
- 물건용도
- 사건·공고번호
- 물건번호 또는 물건관리번호
- 정확한 소재지
- 토지면적(㎡/평)
- 특이사항과 주의사항
- 감정평가액
- 최저매각가격 또는 최저입찰가
- 할인율
- 진행상태와 유찰횟수
- 조회수
- 상세조회 링크

카드 전체를 클릭하면 기존과 동일하게 새 브라우저 탭에서 상세정보와 네이버 지도가 열립니다.

## 사진 표시 방식

1. 온비드 차세대 API의 `thnlImgUrlAdr` 대표사진을 우선 사용합니다.
2. 법원경매 또는 상세 응답에 이미지 URL이 있으면 자동 탐색합니다.
3. 원문 데이터에 사진 URL이 없는 물건은 `물건사진 없음` 자리표시자를 표시합니다.

첫 화면에서 `사진 카드형`과 `상세 표형`을 전환할 수 있으며 기본값은 사진 카드형입니다.
